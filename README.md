# Foodie - Delicious Recipes Website

A modern and responsive website for exploring food recipes from around the world. Built with HTML, CSS, JavaScript, and Bootstrap.

## Features

- **Recipe Search**
  - Search by recipe name
  - Search by first letter

- **Recipe Types**
  - Browse recipes by type
  - View type details and descriptions

- **World Cuisines**
  - Explore recipes by their origin
  - Filter recipes by country/region

- **Ingredients**
  - Browse recipes by main ingredient
  - View detailed ingredient lists

- **Contact Form**
  - User-friendly contact form
  - Real-time form validation
  - Regex-based input validation

- **Recipe Details**
  - Comprehensive recipe information
  - Ingredients and measurements
  - Cooking instructions
  - YouTube video links
  - Source links

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Font Awesome
- TheMealDB API

## API Reference

This project uses [TheMealDB API](https://www.themealdb.com/api.php) for all recipe data. The API is free to use and doesn't require an API key.

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start exploring recipes!

## Features Implementation

### Search
- Real-time search with debouncing
- Search by name or first letter
- Results limited to 20 recipes for better performance

### Navigation
- Responsive side navigation
- Smooth animations
- Mobile-friendly design

### Form Validation
- Name validation (letters and spaces only)
- Email format validation
- Phone number validation (Egyptian format)
- Message length validation
- Real-time validation feedback
